# 🧠 MBTI 성격 유형 진단 테스트

60문항으로 알아보는 나의 MBTI 성격 유형 진단 프로그램입니다.

## ✨ 주요 기능

- 📊 **정확한 진단**: 60개의 상세 질문으로 정확한 MBTI 유형 분석
- 🎨 **컬러풀한 디자인**: 현대적이고 아름다운 UI/UX
- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
- 💾 **PWA 지원**: 앱처럼 설치하여 오프라인에서도 사용 가능
- 🔄 **실시간 진행률**: 테스트 진행 상황을 한눈에 확인
- 📈 **상세한 결과**: 16가지 유형별 특징, 강점, 약점, 추천 직업 제공
- 📤 **결과 공유**: 테스트 결과를 쉽게 공유

## 🚀 시작하기

### 웹 브라우저에서 실행

1. `index.html` 파일을 웹 브라우저에서 엽니다
2. 또는 로컬 서버를 실행합니다:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server 설치 필요)
npx http-server

# PHP
php -S localhost:8000
```

3. 브라우저에서 `http://localhost:8000` 접속

### 모바일 앱처럼 설치 (PWA)

#### iOS (Safari)
1. Safari에서 웹사이트 열기
2. 공유 버튼 탭
3. "홈 화면에 추가" 선택
4. 홈 화면에서 앱 아이콘으로 실행

#### Android (Chrome)
1. Chrome에서 웹사이트 열기
2. 메뉴(⋮) 탭
3. "홈 화면에 추가" 또는 "앱 설치" 선택
4. 앱 서랍에서 실행

## 🎯 MBTI 16가지 유형

### 분석가 (Analysts)
- **INTJ** - 과학자형
- **INTP** - 아이디어 뱅크형
- **ENTJ** - 지도자형
- **ENTP** - 발명가형

### 외교관 (Diplomats)
- **INFJ** - 예언자형
- **INFP** - 잔다르크형
- **ENFJ** - 언변능숙형
- **ENFP** - 스파크형

### 관리자 (Sentinels)
- **ISTJ** - 세상의 소금형
- **ISFJ** - 임금 뒤편의 권력형
- **ESTJ** - 사업가형
- **ESFJ** - 친선도모형

### 탐험가 (Explorers)
- **ISTP** - 백과사전형
- **ISFP** - 성인군자형
- **ESTP** - 수완좋은 활동가형
- **ESFP** - 사교적인 유형

## 🔧 커스터마이징

### 색상 변경
`styles.css` 파일의 `:root` 섹션에서 색상 변수를 수정:

```css
:root {
    --primary-color: #6C5CE7;
    --secondary-color: #A29BFE;
    /* ... */
}
```

### 질문 수정
`questions.js` 파일의 `mbtiQuestions` 배열을 수정하여 질문을 추가/수정/삭제할 수 있습니다.

### 유형 설명 수정
`questions.js` 파일의 `mbtiDescriptions` 객체를 수정하여 각 유형의 설명을 변경할 수 있습니다.

## 💡 사용 팁

### 키보드 단축키
- **1-5**: 답변 선택 (1: 전혀 아니다 ~ 5: 매우 그렇다)
- **←**: 이전 질문
- **→**: 다음 질문

### 정확한 진단을 위한 팁
1. 첫 번째 직관적인 답변을 선택하세요
2. 너무 오래 고민하지 마세요
3. 현재의 자신을 기준으로 답변하세요
4. 이상적인 모습이 아닌 실제 모습을 반영하세요
