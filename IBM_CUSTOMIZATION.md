# IBM 직원 대상 MBTI 테스트 커스터마이징

## 📋 변경 사항 요약

### 1. 질문 개선 (questions.js)
- **질문 수**: 60문항 → 40문항으로 축소
- **질문 내용**: 일반적인 질문 → IBM 직무 환경에 특화된 질문으로 변경
- **각 차원별**: 15문항 → 10문항
  - E/I (외향/내향): 10문항
  - S/N (감각/직관): 10문항
  - T/F (사고/감정): 10문항
  - J/P (판단/인식): 10문항

### 2. 질문 특징
✅ **명확성**: 모호한 질문 제거, 구체적인 업무 상황 중심
✅ **직무 관련성**: IBM 업무 환경과 직접 연관된 질문
✅ **실용성**: 실제 업무 시나리오 기반

#### 예시 질문
- "팀 미팅에서 먼저 의견을 제시하고 토론을 주도하는 것을 선호한다."
- "프로젝트 계획 시 구체적인 단계와 세부 사항을 먼저 정의한다."
- "코드 리뷰나 피드백 시 기술적 정확성을 최우선으로 한다."

### 3. IBM 서비스 추천 기능 추가

각 MBTI 유형별로 적합한 IBM IT 서비스를 1-2개 추천합니다:

#### 추천 서비스 종류
1. **AI & Analytics Service**
   - 데이터 과학, AI/ML 연구, 분석 중심 업무
   - 적합 유형: INTJ, INTP, INFJ, ENTP, ENTJ, ENFJ

2. **Data Platform Service**
   - 데이터 플랫폼 구축, 관리, 최적화
   - 적합 유형: ISTJ, INTJ, INTP, ISTP, ESTJ, ENTJ

3. **Digital Product Engineering iX**
   - 디지털 제품 개발, UX/UI, 사용자 중심 설계
   - 적합 유형: ISFJ, INFJ, ISFP, INFP, ESTP, ESFP, ENFP, ENTP, ESFJ, ENFJ

### 4. MBTI 유형별 설명 개선

각 유형에 대해 IBM 직무 환경에 맞는 설명 추가:
- **IBM Fit**: IBM 내 적합한 역할과 프로젝트
- **Careers**: IBM 관련 직무 추천
- **Recommended Services**: 적합한 IBM 서비스 매칭

#### 예시: INTJ (전략적 설계자)
```javascript
{
    name: "전략적 설계자",
    ibmFit: "클라우드 아키텍처, AI/ML 연구, 엔터프라이즈 아키텍처, 기술 전략",
    recommendedServices: [
        {
            name: "AI & Analytics Service",
            reason: "복잡한 AI 시스템 설계와 데이터 아키텍처 구축에 탁월한 능력을 발휘합니다."
        },
        {
            name: "Data Platform Service",
            reason: "전략적 사고로 확장 가능한 데이터 플랫폼을 설계하고 최적화할 수 있습니다."
        }
    ]
}
```
